Política de Privacidade da Pesquisa

Este é um trabalho acadêmico. Todos os dados coletados serão utilizados exclusivamente para fins de pesquisa, não sendo compartilhados com terceiros nem utilizados para fins comerciais. O anonimato dos participantes é garantido — nenhuma informação pessoal identificável será divulgada.
